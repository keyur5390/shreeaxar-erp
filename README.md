# Shree Axar Furniture Quotation ERP

## Initial Setup
This repository contains the initial setup for the Shree Axar Furniture Quotation ERP.

### Prerequisites
- Node.js
- npm

### Installation
To get started, clone the repository and install the dependencies:

```bash
git clone https://github.com/keyur5390/shreeaxar-erp.git
cd shreeaxar-erp
npm install
```

### Usage
Run the application using:

```bash
npm start
```
